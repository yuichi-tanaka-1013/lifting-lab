# Phase 2 Hypertrophy Block — Upper/Lower 4分割（筋肥大特化）

> Phase 1 LP 卒業後の具体プログラム。筋肥大を主目的、BIG3 は強度維持。
> 制約: 週4回 x 60-90分、[[復帰プログラム: Month 3]] からの継続
> 概念背景は [[筋肥大目的でのフェーズ設計]] を参照（この文書の上位）
> 1RM 目標は [[Strength Roadmap]]、体組成目標は [[Weight Target Ranges]] を参照

## Phase 2 移行タイミング（体調不良反映済）

Week 10 の体調不良 +1週遅延を織り込んだ現実的タイムライン：

| Week | 状態 | アクション |
|------|------|----------|
| Week 11-14 | Phase 1 LP 継続 | ベンチ 70kg、SQ 55-60kg、DL 70-75kg を目指す |
| Week 15-17 | Phase 1 卒業条件確認 + **カット終了**（2026年8月末目標） | 卒業条件達成 + カット終了を同時期に |
| Week 18-19 | Phase 2 準備移行期 | 補助種目の rep を 8-10 → 10-12 にシフト、Isolation 段階導入 |
| Week 20+ | **Phase 2 本格開始**（2026年9月〜） | 下記プログラム適用 |

**Phase 1 卒業条件（3つ全て）**：
1. SQ 60kg 超（Phase 1 の LP を最後まで搾り取った証拠）
2. ベンチ 70kg 到達
3. 週4回が 2-3週連続で安定

**重要な前提**：カット中は Phase 2 に入らない。メンテ or 微増体重前提。
→ 詳細は [[筋肥大目的でのフェーズ設計]] の「出戻り組への示唆」参照

## Phase 2 の設計原則

1. **BIG3 は強度維持のみ**（3x3-5 RIR 1、週1回 per lift）
2. **主軸種目は 6-12 rep**（RIR 1-2）
3. **Isolation は 12-20 rep**（RIR 0-1）
4. **筋群ごとの週セット数を管理**（[[Volume Landmarks]] の MAV レンジ）
5. **rep 進捗優先、重量は最後**（10 → 12 → 15 が揃ってから +2.5kg）

## 週間構成

```
月 Upper A（Push emphasis + Pull maintenance）
火 Lower A（SQ emphasis）
木 Upper B（Pull emphasis + Push maintenance）
金 Lower B（DL emphasis）
土日水 休息 or Daily Home のみ
```

各筋群 週2回頻度。

## 進捗ルール

### 種目別漸進

- **BIG3**：週 +1.25 〜 +2.5 kg 目安（マシン刻み次第）。RIR 1 を守る
- **主軸 compound（8-12 rep）**：rep 進捗優先。10 → 11 → 12 全セット揃ったら +2.5kg
- **Isolation**：rep 進捗優先。12 → 15 → 20 揃ったら +微増（Dumbbell の場合 +2kg、ケーブルの場合 +1ノッチ）

### 週次ボリューム漸進

- 2週間停滞したら 1筋群あたり +1 set/週 追加
- MRV（週20-25 set）に近づいたら Deload 前提

### Deload

- 5-6週おきに 1週間 Deload
- Deload 週: セット数を半分、重量は 60%、RIR 3+

## 中止・後退基準

| 状況 | 対処 |
|------|------|
| BIG3 で RIR が想定より低くなる（RIR 0） | 該当種目 -5% で再スタート |
| 関節痛（肩・肘・膝・腰） | 該当種目を代替に変更（例: ベンチ → DB プレス） |
| 睡眠 6時間切りが2週続く | ボリューム -20%、または Deload 早める |
| 停滞2週続く | 全体 Deload → 次週から +1 set/筋群 |

## Phase 2 の卒業条件（Phase 3 移行判定）

以下が揃ったら Phase 3（ブロック周期化）検討：

1. Phase 2 開始から 6ヶ月経過
2. 主軸種目の rep 進捗が 3週連続で止まる筋群が複数
3. MRV レンジ（週20-25 set / 筋群）で回復が追いつかない
4. InBody で 3ヶ月連続の筋量停滞

→ Phase 3 は Hypertrophy Block と Strength Block を交互に組む。詳細は Phase 2 進行中に別途設計。

## 出戻り組への示唆

1. **BIG3 の伸びは Phase 1 より確実に遅くなる**：Phase 2 は「BIG3 が伸びない」フェーズと覚悟しておく
2. **体組成が主要指標に変わる**：InBody で 3-6ヶ月に 1kg の筋量増を目標
3. **Isolation の導入は「単に種目追加」ではない**：セッション時間が伸びる、集中力配分が変わる、栄養要求が上がる。移行期に慣らす
4. **睡眠・栄養の優先度が上がる**：Phase 1 は「トレーニングでほぼ規定」だったが、Phase 2 は回復要因が支配的
5. **カット中は Phase 2 に入らない**：メンテ or 微増体重が前提。カット中に Phase 2 移行すると volume 追加が回復破綻を招く
