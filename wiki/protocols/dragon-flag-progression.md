# Dragon Flag Progression

> 目標: ドラゴンフラッグをフル ROM で 5 回。
> 腹筋種目の最高難度の一つ。段階的にレベルを上げていく。

## ロードマップ

```
Lv.1 → Lv.2 → Lv.3 → Lv.4 → Lv.5
クランチ  レッグレイズ  ハンギング  ネガティブDF  ドラゴンフラッグ
                     レッグレイズ
（今ここ）                                    （ゴール）
```

## 各レベルの詳細

### Lv.1: クランチ + リバースクランチ（現在）

| 種目 | 回数 | 備考 |
|------|------|------|
| クランチ | 15回 | 肩甲骨が床から離れるだけ。首を引っ張らない |
| リバースクランチ | 12回 | 膝を胸に引きつけ、骨盤を床から持ち上げる |
| デッドバグ | 8/側 | 安定性の基礎。継続 |

**昇格条件**: クランチ 20回 x 3セット + リバースクランチ 20回 x 3セット が**楽に**できる

**推定期間**: 2-3週間

### Lv.2: ライイングレッグレイズ

| 種目 | 回数 | 備考 |
|------|------|------|
| ライイングレッグレイズ | 10-15回 | 仰向け。脚を伸ばしたまま床から90度まで上げて、ゆっくり降ろす |
| クランチ | 20回 | 維持 |
| デッドバグ | 8/側 | 継続 |

**注意**: 腰が床から浮いたら即中止。腰痛既往への配慮。手を臀部の下に敷くと腰が安定する。

**昇格条件**: レッグレイズ 15回 x 3セット、腰を床につけたまま制御できる

**推定期間**: 2-4週間

### Lv.3: ハンギングレッグレイズ（ジムで実施）

| 種目 | 回数 | 備考 |
|------|------|------|
| ハンギングレッグレイズ（脚伸ばし） | 8-10回 | ぶら下がって脚を水平以上に上げる。振れずに制御 |
| ライイングレッグレイズ | 15回 | 維持 |

**注意**: 最初は膝を曲げたバージョンから始めてもOK（ハンギングニーレイズ）

**昇格条件**: 脚を伸ばして 10回 x 3セット、身体が振れずに制御できる

**推定期間**: 1-2ヶ月

**必要な筋力**: グリップ（ぶら下がり耐性）+ 腹直筋下部 + 股関節屈筋

### Lv.4: ネガティブ ドラゴンフラッグ（ジムで実施）

| 種目 | 回数 | 備考 |
|------|------|------|
| ネガティブ ドラゴンフラッグ | 3-5回 | ベンチ台に仰向け。トップポジションから**降ろすだけ**。5秒かけてゆっくり降ろす |
| ハンギングレッグレイズ | 10回 | 維持 |

**やり方**:
1. ベンチ台に仰向け、頭の後ろでベンチの端を掴む
2. 脚を天井に向けてまっすぐ伸ばす（トップポジション）
3. 身体を一直線に保ったまま、**5秒かけてゆっくり降ろす**
4. 降ろしきったら脚を曲げて楽に戻る（上げるのはまだやらない）

**昇格条件**: 5秒ネガティブ x 5回 x 3セット が制御できる

**推定期間**: 1-2ヶ月

### Lv.5: ドラゴンフラッグ（ゴール）

| 種目 | 回数 | 備考 |
|------|------|------|
| ドラゴンフラッグ | 5回 | フル ROM。降ろすだけでなく**上げる動作も含む** |

**やり方**:
1. ベンチ台に仰向け、頭の後ろでベンチの端を掴む
2. 肩甲骨より下を一直線に保ったまま、脚を天井に向けて上げる
3. 一直線を保ったまま、ゆっくり降ろす
4. 床に触れる直前で止めて、再び上げる

**達成 = ゴール**

## 成功に必要な要素

| 要素 | 重要度 | 対策 |
|------|--------|------|
| 腹直筋の絶対的な強さ | ★★★★★ | この Progression で段階的に |
| 体重（軽いほど有利） | ★★★★☆ | カット中（86.8kg → 83-84kg 目標） |
| 股関節屈筋の強さ | ★★★☆☆ | SQ/DL + レッグレイズで強化 |
| 肩・グリップの安定性 | ★★☆☆☆ | ベンチ台を掴む。ラットプルダウン/チンニングで鍛え中 |

## タイムラインの目安

| 時期 | レベル | 種目 |
|------|--------|------|
| 現在（6月） | Lv.1 | クランチ + リバースクランチ |
| 7月中旬 | Lv.2 | ライイングレッグレイズ |
| 8月 | Lv.3 | ハンギングレッグレイズ |
| 9-10月 | Lv.4 | ネガティブ ドラゴンフラッグ |
| 11-12月 | Lv.5 | **ドラゴンフラッグ達成** |
