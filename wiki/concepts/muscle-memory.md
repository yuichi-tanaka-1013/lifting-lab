# Muscle Memory

## 概要

過去にトレーニングで獲得した筋量・筋力を、中断後に初回獲得時より短期間で再獲得できる現象。出戻り組にとって最も重要な味方。科学的にも複数のメカニズムが裏付けられており、「ブロサイエンス」ではなく確立された概念。

## 再獲得速度の推定

[[Greg Nuckols]] の推定（[[SBS: Detraining]] より）:

> **再獲得期間 ≈ 中断期間の約1/2（信頼区間: 1/3〜2/3）**

| 中断期間 | 推定再獲得期間 | 範囲 |
|----------|---------------|------|
| 4週 | 2週 | 1-3週 |
| 12週 | 6週 | 4-8週 |
| 6ヶ月 | 3ヶ月 | 2-4ヶ月 |
| 1年 | 6ヶ月 | 4-8ヶ月 |

### 根拠となる研究

- **Seaborne et al. 2018**: 7週トレ → 7週 detraining → 7週 retraining で元を上回る
- **Henwood & Taffe 2008**: 24週トレ → 24週 detraining → 12週 retraining で完全回復
- **Psilander et al. 2019**: 10週トレ → 20週 detraining → 5週 retraining で元を上回る
- **Ogasawara et al. 2013**: 6週トレ/3週 off の周期的パターンでも continuous group と同等

## 筋核の永続性仮説（Myonuclear Domain）

Muscle memory の中核メカニズム。

1. 筋線維は多核細胞 — トレーニングで筋核（myonuclei）が増加
2. 各筋核は一定範囲の筋線維を管理（= myonuclear domain）
3. **中断で筋線維は萎縮するが、筋核は残存する**（Egner et al. 2013、マウス研究）
4. 再開時: 筋核が oversee する domain が小さい → 転写を迅速に立ち上げ → 再成長が速い
5. Gundersen 2016 レビュー: 筋核が「永久に」残るかは議論があるが、少なくとも筋萎縮と同じ速度では失われない → muscle memory は機能する

## エピジェネティクスの関与

Seaborne et al. 2018:
- hypertrophy に関連する遺伝子（UBR5, TRIM63, SETD3 など）のメチル化パターンが変化
- Detraining 後もメチル化パターンの一部が維持される = 「エピジェネティックな記憶」
- Retraining 時にこれらの遺伝子がより迅速に活性化 → 初回より速い成長

## シグナル経路の再感作

Jacko et al. 2022:
- 長期トレーニングで mTOR 関連シグナル経路が desensitization（鈍化）
- Detraining 期間中に re-sensitization が起こる
- Retraining 時にシグナルがより強く反応 → 初回より強い anabolic response の可能性

## Detraining 後の再獲得速度

→ [[Detraining]] ページの「Retraining における回復速度」も参照

### 復帰プロトコル（Nuckols 推奨）

| 週 | 内容 | 強度 |
|----|------|------|
| 導入週 | 全種目、通常セット数 | 中断前の **1/3 の重量** |
| 第1週 | feel-out | RIR 5+ → RIR 3-4 |
| 第2週〜 | 線形増加 | 週ごとに均等増加 |

- 中断1年超の場合: 初心者プログラムとして再出発を推奨

## 私の実体験（Month 1）

→ 詳細は [[Month 1 Learnings]] を参照

**ベンチプレス 40kg → 60kg を8セッション（4週間）で達成。**

これは本ページの「神経系の回復が先行する」という記述と完全に一致。+50% の重量増加の大部分は motor pattern の再学習と神経系の回復によるものと推定される。

ただし注意点:
- 60kg は baseline 測定時の working weight であり、ピーク時の重量ではない
- 結合組織の回復は筋力の回復に遅れている（ベンチの不安定感が残存）
- 「速く戻った」 = 「元に戻った」ではない

## 出戻り組への示唆

1. **Muscle memory は科学的に裏付けられた現象** — 過去の努力は無駄になっていない
2. ただし **再獲得にも時間がかかる**: 2年8ヶ月の中断 → 推定1年〜1年10ヶ月（中央値: 1年4ヶ月）
3. **神経系の回復が先行**: 最初の数週で「急に強くなった」と感じるのは motor pattern の再学習
4. 筋量の回復はそれに遅れる → 焦って重量を追わない
5. **結合組織は muscle memory の恩恵を受けにくい**: 筋力回復 >> 結合組織回復 → 怪我リスクの非対称性
6. 周期的トレ/中断パターン（Ogasawara 2013）の知見は、今後忙しい時期に「3週 off」が入っても致命的ではないことを示す
