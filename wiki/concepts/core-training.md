# Core Training

## 概要

「体幹トレーニング」は2つの異なる目的で行われる。混同しないこと。

### 1. 安定性（Stability）— パフォーマンス目的

- 腹横筋、多裂筋などの深層筋
- スクワット・デッドリフト時に腰椎を守る「コルセット」
- 種目: デッドバグ、プランク、パロフプレス
- **高レップ・低負荷・制御重視**

### 2. 筋肥大（Hypertrophy）— 見た目目的

- 腹直筋（シックスパック）、腹斜筋（くびれ）
- カットで脂肪が落ちた時に「見せる腹筋」があるかどうか
- 種目: クランチ、リバースクランチ、レッグレイズ、ドラゴンフラッグ
- **漸進性過負荷を適用する**（回数↑ → 難易度↑ → 加重）

## 腹筋が見えるための2条件

```
見える腹筋 = 低い体脂肪率（~15%以下） × 十分な腹直筋の厚み
```

どちらか一方だけでは不十分:
- 体脂肪率が低くても腹筋が薄い → 「痩せた腹」（ガリガリ）
- 腹筋が厚くても体脂肪率が高い → 「隠れた腹筋」（パワーリフター体型）

## 腹直筋の解剖

| 部分 | 通称 | 効果的な種目 |
|------|------|------------|
| 上部 | シックスパックの上4つ | クランチ、ケーブルクランチ |
| 下部 | シックスパックの下2つ | リバースクランチ、レッグレイズ、ドラゴンフラッグ |
| 腹斜筋 | くびれ・側面 | バイシクルクランチ、サイドプランク |

> 厳密には腹直筋は1つの筋肉で「上部/下部」に分離して鍛えることはできないが、
> 種目によって刺激の比重が変わるため、実践的には上部/下部を意識した種目選択に意味がある。

## 腹筋トレーニングの頻度

- 腹筋は他の筋群より回復が速い → **毎日でも可能**
- ただし高強度（加重クランチ、ドラゴンフラッグ等）は週3-4回が適切
- 低強度（デッドバグ、プランク）は毎日OK

## 漸進性過負荷の適用

腹筋にも [[Progressive Overload]] は適用される:

1. **レップ数を増やす**（15 → 20 → 25）
2. **セット数を増やす**（2 → 3）
3. **種目の難易度を上げる**（クランチ → レッグレイズ → ドラゴンフラッグ）
4. **加重する**（加重クランチ、ケーブルクランチ）

## 出戻り組への示唆

- カット中に腹筋トレを並行することで、脂肪が落ちた時に「割れた腹」が出現する
- 腹筋の筋肥大はカロリー赤字下でも起こりやすい（小さい筋群のため回復負荷が低い）
- スクワット・デッドリフトでも腹筋は使われるが、「見た目の腹筋」にはアイソレーションが必要
