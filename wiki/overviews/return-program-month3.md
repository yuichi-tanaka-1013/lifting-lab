# 復帰プログラム: Month 3（Week 9〜）

> Month 2 を完了し、下半身の導入・ベンチの安定化・Upper/Lower 4分割への移行が成功。
> Month 3 では LP を継続しつつ、Phase 2 への移行準備とアイソレーション種目の段階的導入を行う。

## Month 2 → Month 3 の引き継ぎ

### 達成したこと（Month 2）

| 項目 | 結果 |
|------|------|
| BB スクワット | 0 → 40kg（baseline 到達） |
| BB デッドリフト | 40kg → 55kg（baseline 60kg にあと1週） |
| ベンチプレス | 60kg 潰れ → 57.5kg 安定化 → 60kg 復帰 → 62.5kg 自己ベスト更新 |
| ラットプルダウン | 59kg レップ崩壊 → 54kg で安定 → 54.3kg x 10 x 3 達成 |
| プログラム形式 | フルボディ 2回/週 → Upper/Lower 4回/週 |
| 計画遵守 | Week 5 で3回逸脱 → Week 6 以降は全セッション遵守 |

### 持ち越し課題

| 課題 | 対応 |
|------|------|
| ベンチの不安定感 | 改善傾向だが継続監視。RC プリハブは毎回必須 |
| Phase 2 への移行判定 | LP が止まる兆候を監視（種目ごとに判断） |
| 腕のアイソレーション導入 | Week 10-12 で条件確認後に正式導入 |
| 体脂肪カット | 継続中（-500 kcal/日、トレ日 米1.8合/休息日 米1.2合） |
| InBody 再測定 | Week 10-11 の週末に予定 |

## プログラム形式

**Upper/Lower 4分割（月火木金）+ 日によって +30min オプション**

| 日 | セッション | 時間 |
|----|----------|------|
| 月 | Upper A（プッシュ重点） | 60分 / 90分 |
| 火 | Lower A（スクワット） | 60分 / 90分 |
| 水 | 休み（ウォーキング推奨） | — |
| 木 | Upper B（プル重点） | 60分 / 90分 |
| 金 | Lower B（ヒンジ） | 60分 / 90分 |

- 60分のメインメニューは単独で完結する
- 90分取れる日は +30min オプションを追加。やらなくても問題ない
- オプションの実施可否は日によってバラバラなので、チートシートに毎日記載

### LP 停滞（Phase 2 移行）の判定基準

以下の兆候が出たら、その種目は Phase 2 に移行:

1. 同じ重量で RIR が前週より下がった
2. 目標 rep に届かないセッションが **2回連続**
3. **2週連続で重量据え置き**

→ 種目ごとに判断。ベンチが Phase 2 に入っても、スクワットは LP を継続する可能性あり。

### Phase 2 の移行先（予定）

| 手法 | 内容 |
|------|------|
| DUP（Daily Undulating Periodization） | 同じ種目で日によってレップ数を変える（例: 月 5x5 重い / 木 3x10 軽い） |
| ボリューム増加 | 3セット → 4セット |
| バリエーション追加 | インクラインDB、クローズグリップベンチ等 |
