# 現在の筋量から見た BIG3 ポテンシャル（2026-07 時点）

> Query「私の今の筋肉量でビッグスリーのポテンシャルを推測してください」への回答を保存。
> **結論: 現在の筋量 (LBM 70.1kg / SMM 40.3kg) は、神経系・結合組織の再適応が完了すれば BIG3 合計 300-350kg を支えられる。今の合計 ~240kg とのギャップ約 60-110kg は筋量不足ではなく再適応の途中であることによる。**

## 推定: 現在の筋量が支えられる 1RM

### 根拠1: 自分自身の実績（最も信頼できる証拠）

ピーク時 (2023-09) の BIG3 合計は **~300 kg**（[[Strength Roadmap]]）。当時の SMM 40.9 kg に対し現在 40.3 kg — **差はわずか 1.5%**。つまり**ほぼ同じ筋量で 300 kg を実際に挙げた実績がある**。これは推定ではなく実測。

### 根拠2: FFMI / 体重比の一般水準

FFMI 21.3・86 kg 級の鍛錬済みナチュラルの典型到達域（Symmetric Strength / ExRx 系の中級〜上級下限）:

| 種目 | 典型域 | 現在の筋量での見立て |
|------|-------|-------------------|
| ベンチ | 1.1-1.4x BW | **95-110 kg** |
| スクワット | 1.4-1.7x BW | **110-135 kg** |
| デッドリフト | 1.6-2.0x BW | **135-160 kg** |
| **合計** | 4.1-5.1x BW | **340-405 kg**（上限側は要・数年の専門的トレ） |

### 統合結論

| 段階 | BIG3 合計 | 内訳目安 | 必要条件 |
|------|----------|---------|---------|
| 現在 | ~238 kg | 90 / 65 / 83 | — |
| **筋量そのまま・再適応完了** | **300-350 kg** | ~95-105 / ~100-120 / ~130-145 | 神経系・結合組織の回復（筋量追加は不要）。Muscle Memory 圏内 |
| 筋量そのままの理論上限付近 | ~350-380 kg | | 数年の specificity（挙上技術・ピーキング） |
| 上級ライン 470 kg | | 120 / 160 / 190 | **FFMI ~23（LBM +5-6 kg）が必要 = Phase 2 以降の筋肥大** |

## 種目別の所見

- **ベンチ (~90)**: 3種目で最も回復が進んでいる。体幹・腕の筋量はピーク比 -3〜-6% だが既に中級ライン (90kg) に到達目前。現筋量での天井は 95-110。**伸び代は残っているが、3種目では最も天井が近い**
- **スクワット (~65)**: **最大のギャップかつ最大の伸び代**。脚 SMM 20.95kg は 6週で +0.26kg と唯一増加中の部位で、現在の 1RM 0.76x BW は筋量ではなく動作再学習・結合組織がボトルネック。現筋量で 110-135 は十分視野
- **デッドリフト (~83)**: SQ 同様ヒンジ再学習の途中。骨ミネラル量がピーク超え (3.92kg) しており、荷重耐性の土台は既に整っている。現筋量で 135-160

## 諸説あり

- FFMI・体重比と 1RM の相関は集団レベルでは強い (r~0.8) が、**個人差 ±15% 程度**（四肢長・モーメントアーム・筋線維タイプ・腱付着位置）。本推定のレンジ幅はこれを反映
- 「中級/上級」の体重比基準はソースにより異なる（[[Strength Roadmap]] の諸説ありと同様）
- InBody の SMM は DEXA 比で誤差あり。ただし同一機器の経時比較なので傾向は信頼できる

## 出戻り組への示唆

1. **「300kg までは筋量の借金なしで戻れる」が本推定の核心**。焦って重量を追わなくても、LP を回し続ければ筋量側の裏付けは既にある
2. ⚠️ ただし**この数字を毎週の作業重量のアンカーにしないこと**（[[Current State]] のリスクマップ通り、結合組織の回復は筋力より遅い）。ポテンシャル ≠ 今週挙げていい重量
3. SQ/DL の伸び代が大きい = 今後数ヶ月の LP で合計が最も速く伸びるのは下半身。ベンチの停滞を感じてもプログラム全体は前進している
4. 470kg (上級) は現筋量では届かない。Phase 2 以降の筋肥大 (+5-6kg LBM) とセットの長期目標
