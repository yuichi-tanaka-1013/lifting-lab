## Rep レンジと目的の対応

エビデンスベース（[[Brad Schoenfeld]] 2017 meta、[[Eric Helms]] MASS レビュー系）：

| Rep レンジ | 主目的 | 神経系負荷 | ボリューム稼ぎ | フォームリスク |
|-----------|-------|----------|-----------|-----------|
| 1-3 reps | 神経系適応（強度） | 極高 | 低 | 高 |
| 4-6 reps | 強度＋筋肥大 hybrid | 高 | 中 | 中 |
| 6-12 reps | **筋肥大の主軸** | 中 | 高 | 低 |
| 13-20 reps | 筋肥大＋メタボリック | 低 | 高 | 極低 |
| 20+ reps | 補助・耐性 | 極低 | 中 | 極低 |

### 重要な事実

- **筋肥大は 5-30 rep で同等に起こる**（総ボリュームが揃えば）
- **効率で見ると 6-12 rep が sweet spot**
- **3-rep 中心は strength lifter 向け**。筋肥大目的だとコスト過剰

## 筋肥大目的での rep 配分（週全体で見る）

| Rep レンジ | 全体の割合 | 割り当てる種目 |
|-----------|-----------|-------------|
| 3-5 reps | 10-20% | BIG3 の強度基盤 |
| **6-12 reps** | **50-70%** | **主要 compound + 補助種目** |
| 13-20 reps | 20-30% | Isolation |
| 20+ reps | 少量 | カーフ、アブス |

## 3-Phase モデル

### Phase 1: LP（Linear Progression）— 現在

**期間**: 3-4ヶ月（Week 1-17 想定、体調不良で +1週）
**目的**: baseline 復帰、神経系再構築、フォーム再学習
**詳細プログラム**: [[復帰プログラム: Week 1-4]], [[復帰プログラム: Week 5-8]], [[復帰プログラム: Month 3]]

**構成**:
- BIG3: 3x5（強度基盤）
- 補助 compound: 3x8-10
- Isolation: 少量 or Option 枠のみ
- 頻度: 週4回 Upper/Lower

**特徴**: muscle memory が働くフェーズ。強度も筋量も速く戻る。

**卒業条件（3つ全て）**:
1. SQ/DL が baseline 60kg 超
2. ベンチが 65-70kg 到達
3. 週4回が 2-3週連続で安定

**卒業後の即時アクション**: アイソレーション正式導入 → Phase 2 準備期 → Phase 2

### Phase 2: Volume Block（筋肥大加速）— 次

**期間**: 6-12ヶ月（Week 18+ 想定）
**目的**: 筋量を積極的に増やす。強度基盤は維持。
**詳細プログラム**: [[Phase 2 Hypertrophy Block]]

**構成の変化**:

| 項目 | Phase 1 | Phase 2 |
|------|--------|--------|
| BIG3 の rep | 3x5 | **3x3-5**（強度維持、週1回 per lift） |
| 補助 compound の rep | 3x8-10 | **3x8-12**（週2回 per muscle） |
| Isolation | Option 少量 | **本格導入**（2-3x12-20 RIR 0-1） |
| 頻度 | 週4回 | 週4回維持（強度は分散） |
| 進捗指標 | BIG3 の kg | 主軸種目の rep 進捗 + 筋群別週セット数 + 体組成 |
| Volume 管理 | 意識薄 | **[[Volume Landmarks]]（MEV/MAV/MRV）** |

**Phase 2 内の DUP 要素**:
- 各 lift を「重い日（3x3-5 RIR 1）」+「軽い日（3x8-12 RIR 2）」の週2回
- これが筋肥大＋強度維持の両立解

**卒業条件（Phase 3 移行）**:
1. Phase 2 開始から 6ヶ月経過
2. 主軸種目の rep 進捗が 3週連続で止まる筋群が複数
3. MRV レンジで回復が追いつかない
4. InBody で 3ヶ月連続の筋量停滞

### Phase 3: Block Periodization — 将来

**期間**: Phase 2 で 6-12ヶ月経過後
**目的**: 長期プラトー回避
**詳細プログラム**: 未設計（Phase 2 中に設計）

**構成**:
- 4-6週 Hypertrophy Block（rep 8-15、RIR 0-2、MAV-MRV）
- 3-4週 Strength Block（rep 3-6、RIR 1-3、Peak）
- 1週 Deload

## 出戻り組への示唆

1. **Phase 1 は muscle memory で速く卒業できる**：3-4ヶ月が目安
2. **BIG3 の伸びは Phase 1 で最も速い**：Phase 2 以降は減速。これが正常
3. **Phase 2 でボリューム管理を学ぶ**：Phase 3 の準備
4. **過去のスタイル（strength 型）と現在の目的（筋肥大）を分ける**：ピーク時 100kg x 10 DL は strength lifter 型。筋肥大目的なら rep レンジと種目選択を変える
5. **BIG3 の数字は目的ではなく手段**：Phase 2 以降、BIG3 の伸びは減速する。それは正常
6. **体組成計測（InBody）が Phase 2 の主要指標**：筋量ベースの進捗を追う
7. **カット中は Phase 2 に入らない**：メンテ or 微増体重が前提。カット終了 + Phase 1 卒業を同時期にする

## 諸説あり（両論併記）

### 高頻度 vs 中頻度

- 高頻度派（各筋群 週3回）：Norwegian Frequency Project、Nippard 等
- 中頻度派（各筋群 週2回）：Schoenfeld 2016 meta（週2 = 週3、ただし週1 < 週2）
- **保守的な結論**：週2回で十分。頻度上げは volume 増より効果薄

### 低ボリューム vs 高ボリューム

- 低ボリューム派（週6-10 set / 筋群）：[[山本義徳]] 等
- 高ボリューム派（週15-25 set / 筋群）：Israetel、Nippard
- **保守的な結論**：MEV（10-12）から始めて反応見て増やす

### RIR 0 vs RIR 1-2

- 追い込み派（RIR 0）：Compound には不向き、isolation には有効
- 余裕派（RIR 1-2）：Compound の主流。回復と volume の balance
- **保守的な結論**：Compound は RIR 1-2、Isolation は RIR 0-1
