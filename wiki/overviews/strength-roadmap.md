# Strength Roadmap — 中級者・上級者への道

> 体重 80kg（カット後の目標体重）を基準とした筋力レベルのロードマップ。

| 種目 | 現在の推定 1RM | 体重比 |
|------|-------------|--------|
| ベンチ | ~82 kg (65kg x 8) | 1.03x |
| スクワット | ~52 kg (45kg x 5) | 0.65x |
| デッドリフト | ~70 kg (60kg x 5) | 0.88x |
| **合計** | **~204 kg** | **2.55x** |

## ロードマップ全体像

```
現在          中級者            上級者
204 kg ────→ 350 kg ─────────→ 470 kg
2026.06       2026.10           2027-2028

 Phase 1        Phase 1→2          Phase 2→3
 LP(線形進行)    LP終了→DUP等        周期化・ピーキング
 +2.5-5kg/週    +2-5kg/月          +1-2kg/月
```

## Phase 別の戦略

### Phase 1: LP（今〜2026年10月頃）

**やること**: 毎週 +2.5-5kg を積み上げる。今やっていることをそのまま継続。

| 種目 | 到達見込み（10月） | 備考 |
|------|----------------|------|
| ベンチ | 80-90 kg x 8 (1RM ~100kg) | **ピーク時に復帰** |
| スクワット | 100-110 kg x 5 (1RM ~120kg) | **ピーク時を大幅に超える** |
| デッドリフト | 120-130 kg x 5 (1RM ~140kg) | **ピーク時に復帰** |

LP が止まる時期は種目によって異なる:
- **ベンチ**: 70-80kg あたりで鈍化の可能性（あと 4-6 週）
- **スクワット**: まだ伸びしろ大。80-100kg あたりまで LP が続く可能性
- **デッドリフト**: 同上。100-120kg あたりまで

### Phase 2: 中級者プログラム（2026年秋〜2027年）

LP が止まったら移行。月単位で +2-5kg。

**プログラムの変化:**

| 項目 | Phase 1（今） | Phase 2 |
|------|-------------|---------|
| 重量増加 | 毎週 | 月 2-5kg |
| レップスキーム | 固定（5 or 8） | DUP: 重い日 5x5 / 軽い日 3x10 |
| セット数 | 3 | 3-5 |
| 周期化 | なし | 3週追い込み + 1週 deload |
| 補助種目 | 少ない | 増える（弱点部位の強化） |

**中級者で追加される種目の例:**

| 種目 | 目的 |
|------|------|
| インクラインDBベンチ | ベンチの補助（上部胸筋） |
| フロントスクワット | スクワットの補助（四頭筋・コア） |
| RDL（重い） | デッドリフトの補助（ハムストリング） |
| バーベルロウ | 背中のボリューム増加 |
| チンニング（加重） | ラットプルダウンからの卒業 |

### Phase 3: 上級者プログラム（2027年〜）

月単位で +1-2kg。ここからは「プログラミングの質」が差を生む。

| 要素 | 内容 |
|------|------|
| ブロック周期化 | 筋肥大期 → 筋力期 → ピーキング期 をサイクル |
| RPE/RIR の精密管理 | セットごとに調整 |
| 弱点分析 | スティッキングポイントの特定 → 補助種目で対策 |
| 栄養の最適化 | バルク/カット のサイクル管理 |
| 回復の最適化 | sleep, ストレス管理がより重要に |

## 各マイルストーンの意味

| マイルストーン | 合計 | 時期 | 意味 |

| **ピーク復帰** | ~300 kg | **2026年8-9月** | 2023年の自分に戻る |

| **中級者** | ~350 kg | **2026年10月** | 「ジムで強い方」に入る |

## 体重の影響

カットで 87kg → 80kg に落とすと:

| 効果 | 詳細 |
|------|------|
| 体重比が上がる | 同じ 1RM でもランクが上がる |
| 相対筋力が上がる | 懸垂が楽になる、ドラゴンフラッグが近づく |
| 見た目が変わる | 筋肉のカットが出る |
| 絶対筋力は微減の可能性 | ただしリコンプ中なら影響は最小限 |

**80kg / 体脂肪率 15% で BIG3 合計 350kg** が中級者の入り口。これが「健康 × 動ける × 見た目」の全てを満たすポイント。
