# 復帰プログラム: Week 1-4 — Plan vs Actual

> **Confidence: medium** — 個人の状態に合わせた設計であり、エビデンスに基づく原則を適用しているが、
> 個人プログラムとして最適かは実際のトレーニングでの反応を見て調整が必要。
> 特に肩の状態は週単位でモニタリングし、悪化があれば即座に種目変更すること。

> **注記（2026-05-18）**: このプログラムは Plan として設計されたが、実際には本人が独自に1ヶ月間の
> トレーニングを先行して実施していた。以下の「Plan vs Actual」セクションで計画と実績を比較する。
> 詳細は [[Month 1 Learnings]] を参照。

## Plan vs Actual: 4週間の比較

### 重量推移

| 種目 | Plan (W1→W4) | Actual (W1→W4) | 評価 |
|------|-------------|----------------|------|
| ベンチ系 | DB 14kg→20kg/手 | **BB 40kg→60kg** | 種目が異なる。BBベンチで実施し、計画より速い進行 |
| スクワット | 40kg→55kg | **未実施** | **大きな乖離** |
| デッドリフト | 40kg→55kg | **未実施** | **大きな乖離** |
| ラットプルダウン | 軽め→漸増 | 40kg→54kg x 10 | 良好な進行 |

### 構造の比較

| 項目 | Plan | Actual | 評価 |
|------|------|--------|------|
| プログラム形式 | フルボディ A/B | 上半身プッシュ/プルのみ | 乖離 — ただし継続性は確保 |
| 頻度 | 週2-3回 | 週2回（全4週） | 許容範囲 |
| ベンチ種目 | DBベンチ（RC配慮） | BBベンチ | **計画と逆の選択** |
| RC プリハブ | 毎回必須 | 不明（おそらく不十分） | 要改善 |
| RIR管理 | 6+→3-4 で漸進 | 不明（感覚ベース） | ログ不足 |
| MV フォールバック | 3パターン設計 | 使用機会なし | 保険として有効 |

### 計画が正しかった点

1. **開始重量 40kg**: 実際にも 40kg から開始しており、エビデンスベースの設定が本人の直感と一致
2. **週2回でも十分に進行可能**: 計画の「パターン1: 週2回」と実際の頻度が一致し、成長は確認された
3. **RC の違和感は低負荷再導入で改善**: 計画の前提通り、最初の2週間の軽い痛みを経て改善傾向

### 計画が外れた点と学び

1. **BBベンチの「禁止」は厳しすぎた**: 実際にはBBベンチで開始し、RC は悪化せず改善した。ただし**不安定感は残存**しているため、RC プリハブの追加は依然として推奨
2. **フルボディは心理的ハードルが高かった**: 「上半身のみ」で再開した方が継続のハードルが低く、全4週を途切れなく実施できた。理論上の最適 ≠ 実行可能性の最適
3. **下半身の遅れはコスト**: 上半身に集中した1ヶ月は継続面では成功だが、下半身は完全に0からスタートが必要。この「借り」は Month 2 で返す必要がある

### 禁止事項の振り返り

| 禁止事項 | 遵守 | 結果 |
|----------|------|------|
| 1RMテスト / RIR 0-1 | たぶん遵守 | 怪我なし |
| BBベンチ | **未遵守** | RC改善したが不安定感残存。結果オーライだが、リスクは取っていた |
| 「昔の重量」基準の漸進 | 遵守 | 40kgから丁寧にスタート |

### 私の現在地（[[Current State]] より）

| 項目 | 値 |
|------|-----|
| 推定1RM ベンチ | 78 kg（60kg x 9） |
| 推定1RM スクワット | 70 kg（60kg x 5） |

| トレーニング頻度 | 週2-3回、60-90分 |

### プログラム形式

**フルボディ A/B 交互** — [[Full Body for Returning Lifters]] に準拠

- 週2回: A → B（中2-3日）
- 週3回: A → B → A → 翌週 B → A → B...（中1-2日）
- 全身を毎回刺激することで、限られた頻度で各筋群に週2-3回のシグナルを送る

### 重要な種目選択の判断

**ベンチプレスをバーベルではなくダンベルで行う理由:**

肩のローテーターカフに違和感があり、バーベルベンチで不安定感がある状態（[[Current State]] リスクマップ）。ダンベルベンチは:
- 肩関節が自然な軌道で動ける（バーベルは固定軌道を強制する）
- 可動域を個人に合わせて調整可能
- 左右差の検出と修正が可能
- グリップ角度を微調整できる（ニュートラル寄りにすることでRC負荷を軽減）

バーベルベンチへの移行は、肩の違和感スコアが安定して 1-2/5 以下になってから検討する。

### Week 2 で特に注意すべきこと

[[SBS: Detraining]] からの引用:

> セット間でパフォーマンスが予測不能に変動する可能性がある。
> 急速な疲労（デコンディショニング）が起きたり、
> 逆に motor pattern の再学習で後半セットが楽になることもある。

- 1セット目が想定より重ければ → 即座に重量を落とす
- 1セット目が想定より軽ければ → 重量を上げてもよいが RIR 4 以下にしない

## 漸進性過負荷ルール（[[Progressive Overload]]）

### 増やす条件

| 条件 | 判定 |
|------|------|
| 前週の全セットで目標RIRを達成した | YES → 増加可能 |
| 肩・腰・手首に新たな違和感がない | YES → 増加可能 |
| 翌日〜翌々日の DOMS が許容範囲（日常生活に支障なし） | YES → 増加可能 |
| **3つすべて YES** | **→ 重量を増やす** |
| **1つでも NO** | **→ 据え置き（同重量を繰り返す）** |

### 増加量

| カテゴリ | 増加幅/週 | 根拠 |
|----------|----------|------|
| スクワット/デッドリフト | **+5 kg** | [[SBS: Detraining]]: 下半身 +5-10lb/週 |
| DBベンチ/DBインクライン | **+2 kg/手** | [[SBS: Detraining]]: 上半身 +2.5-5lb/週 |
| ケーブル/マシン | **+1ノッチ** | feel ベースで調整 |

### 増やさない判断をした場合

- 同じ重量・セット・レップを次週も繰り返す
- 2週連続で据え置きになったら → 何かが間違っている。以下を確認:
  1. 睡眠時間（5時間以下が続いていないか）
  2. 食事（タンパク質が不足していないか）
  3. 肩/腰/手首の状態（痛みを無視していないか）
  4. ストレスレベル（仕事の変化はないか）

## 絶対にやってはいけないこと 3つ

### 1. 1RMテストおよび RIR 0-1 のセットを行うこと

**理由**: 結合組織が2年8ヶ月間デコンディショニングされている（[[Detraining]]）。筋力は [[Muscle Memory]] で速く回復するが、腱・靭帯は追いつかない。最大努力の挙上は、筋力が結合組織の耐性を超えるリスクがある。特にローテーターカフの違和感がある状態での max effort ベンチは、肩の怪我に直結する。

> [[SBS: Detraining]]: 「低ボリューム高重量で筋力だけ急回復させるアプローチは避けるべき。
> 腱・結合組織がデコンディショニングされたまま高重量を扱うことになる」

**最初の4週間は RIR 3 を下限とする。**

### 2. ベンチプレスをバーベルで行うこと（この4週間は）

**理由**: 肩関節でベンチが不安定に感じる + ローテーターカフの違和感（[[Current State]]）。バーベルベンチはバーの軌道が固定されるため、肩関節に不自然な負荷がかかりやすい。ダンベルで肩の安定性と可動域を再構築してから、バーベルに移行する。

**例外**: スミスマシンでの低負荷ベンチ（ウォームアップの一環として RIR 8+ で）は許容範囲。ただし working set にはしない。

### 3. 「昔の重量」を基準に漸進ペースを決めること

**理由**: アンカリングバイアス（[[Current State]], [[Detraining]]）。ピーク時の重量は **今の身体とは無関係**。漸進は今週の RIR と身体のフィードバックのみで判断する。「前は○○kg 挙がってたのに」という思考が浮かんだら、それは muscle memory が仕事をしている途中であり、結合組織はまだ追いついていないという事実を思い出す。

## Week 5 以降への橋渡し

Week 4 終了時に以下を判断:

1. **肩の状態**: 改善 → バーベルベンチの段階的導入を検討 / 変化なし → DBベンチ継続 / 悪化 → 種目見直し
2. **線形進行の継続可能性**: RIR が目標範囲内 → 継続 / RIR が下がってきた → ボリューム or 頻度の調整
3. **ボリュームの増加**: 各筋群週9セット → 12セットへの段階的増加を検討
4. **チンニング**: バンドアシストで5回できるようになったら、メイン種目に格上げ
