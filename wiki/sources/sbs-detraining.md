# SBS: Detraining — What Happens When You Stop Training?

## 記事の核心

トレーニング中断がパフォーマンスと筋量に与える影響を、Bosquet 2013 メタアナリシス（103研究）を軸に体系的にレビュー。Muscle memory のメカニズム、中断中の損失緩和策、復帰プロトコルまでを一貫してカバー。

## 主要な知見

### 1. 中断による筋力低下のタイムライン

- **28日まで**: 若年成人では最大筋力への影響は軽微（g < 0.2）
- **28日以降**: 低下が加速
- 高齢者（65歳以上）は若年成人の約2倍の速度で低下
- 筋持久力は最大筋力より早く低下（代謝的適応の喪失が加算されるため）

### 2. 筋量低下

- 全筋サイズ（thickness, CSA）は detraining で低下
- ただし筋線維断面積（fiber CSA）は比較的維持 → 見た目の変化 > 実質の筋量変化の可能性
- 実践的推定: 筋量低下は筋力低下とおおよそ並行

### 3. Muscle Memory

- **再獲得期間 ≈ 中断期間の約1/2**（信頼区間: 1/3〜2/3）
- メカニズム: 筋核永続性、エピジェネティクス、mTOR 再感作
- Ogasawara 2013: 6週トレ/3週 off の周期パターンでも continuous と同等の成果

### 4. 最小維持ボリューム（Bickel et al. 2011）

- **若年成人は元のボリュームの 1/9 で8ヶ月間、筋量・筋力を維持可能**
- 週1-2回、30-45分の自重トレでも大部分を維持できる

### 5. 復帰プロトコル

- 第1週: 全種目を通常セット数で、ただし重量は中断前の 1/3
- 第2週: RIR 3-5 で feel-out
- 第3週以降: 線形増加（中断前重量との差分 ÷ 推定回復週数）
- 中断1年超: 初心者プログラムとして再出発を推奨
- **避けるべき**: 低ボリューム高重量での急速筋力回復（結合組織が追いつかない）

## 出戻り組への示唆

1. **2年8ヶ月の中断** → 記事の分類では「1年超」に該当し、初心者プログラムでの再出発が推奨される
2. **Muscle memory は確実に味方**: ただし再獲得に要する期間は中断の長さに比例するため、数ヶ月単位の視野が必要
3. **子育てによる中断**は記事が挙げる「時間不足」カテゴリに合致 — 今後の中断予防として最小維持ボリュームの知識が直接使える
4. **復帰第1週は中断前の1/3重量**: 現在の推定1RMからさらに下げるのではなく、中断前ベースで計算する点に注意
5. **結合組織のデコンディショニング**: 記事が最も強く警告しているのは、筋力だけ急回復させて結合組織が追いつかないパターン → 肩のローテーターカフの問題と直結
