# Week 6 Cheat Sheet — Upper/Lower 4分割

> **書いてある重量を守る。「もっといける」は muscle memory であって結合組織の準備完了ではない。**

---

## 月: Upper A（プッシュ重点）

### Warm-up（7分）

| 種目 | 回数 |
|------|------|
| 早歩き or ジャンピングジャック | 2分 |
| バンド外旋 | 15 x 2 |
| ショルダーディスロケーション | 10回 |
| バンドプルアパート | 15回 |
| ベンチ バーのみ | 10回 |

### Main

| 種目 | 重量 | セット x レップ | RIR |
|------|------|----------------|-----|
| ベンチプレス | **57.5 kg** | 3 x 8 | **3** |
| ケーブルロウ | **45 kg** | 3 x 10 | 3 |
| DBショルダープレス | **12 kg x 2** | 3 x 10 | 4 |
| フェイスプル | 9.6 kg | 2 x 15 | — |

**NG**: ベンチ 60kg にしない。3セット x 8rep を RIR 3 で揃えることが今週の課題。

## 火: Lower A（スクワット）

### Warm-up（10分）

| 種目 | 回数 |
|------|------|
| バイク or 早歩き | 2-3分 |
| グルートブリッジ（トップ2秒） | 10 x 2 |
| デッドバグ | 6/側 x 1 |
| ウォールアンクルストレッチ | 30秒/側 |
| ボディウェイトSQ（ボトム3秒） | 5回 |
| バー x 5（SQ ウォームアップ） | 1セット |

### Main

| 種目 | 重量 | セット x レップ | RIR |
|------|------|----------------|-----|
| BB スクワット | **30 kg** | 3 x 5 | **6+** |
| レッグプレス | 軽め | 3 x 10 | 4 |
| レッグカール | 軽め | 3 x 10 | 4 |
| プランク | 自重 | 2 x 30秒 | — |

**NG**: スクワットを 30kg 以上にしない。BB 初回はフォーム最優先。

### Form Check（BB スクワット）

- [ ] パラレル以下まで降りている
- [ ] 膝がつま先方向
- [ ] 背中ニュートラル
- [ ] 踵が浮いていない
- [ ] バーが安定している
