## Day A（スクワット導入）

### Warm-up（8分）

| 種目 | 回数 |
|------|------|
| バイク or 早歩き | 2分 |
| グルートブリッジ（トップ2秒） | 10 x 2 |
| デッドバグ | 6/側 x 1 |
| ウォールアンクルストレッチ | 30秒/側 |
| ボディウェイトSQ（ボトム3秒） | 5回 |
| バンド外旋 | 15 x 2 |

### Main（重量は全てRIR記載の通りに抑える）

| 種目 | 重量 | セット x レップ | RIR |
|------|------|----------------|-----|
| **ゴブレットSQ** | **10 kg** | 3 x 8 | **8+** |
| ベンチプレス | 60 kg | 3 x 8 | 3 |
| ケーブルロウ | 前回+α | 3 x 10 | 3 |
| フェイスプル | 軽め | 2 x 15 | — |
| プランク | 自重 | 2 x 30秒 | — |

**NG**: ゴブレットSQ を「軽すぎる」と感じて重量を上げない。10kg は意図的。

### Form Check（ゴブレットSQ、横から動画撮影推奨）

- [ ] パラレル以下まで降りている
- [ ] 膝がつま先方向（内側に入らない）
- [ ] 背中ニュートラル（butt wink なし）
- [ ] 踵が浮いていない
- [ ] ボトムで2秒止まれる

## Day B（ヒンジ導入）

### Warm-up（8分）

| 種目 | 回数 |
|------|------|
| バイク or 早歩き | 2分 |
| グルートブリッジ（トップ2秒） | 10 x 2 |
| デッドバグ | 6/側 x 1 |
| ヒップサークル | 8/方向 |
| キャット&カウ | 10回 |
| バンド外旋 | 15 x 2 |

### Main

| 種目 | 重量 | セット x レップ | RIR |
|------|------|----------------|-----|
| **DB RDL** | **10 kg x 2** | 3 x 8 | **8+** |
| ベンチプレス | 60 kg | 3 x 8 | 3 |
| ラットプルダウン | 56 kg | 3 x 10 | 3 |
| バンドプルアパート | バンド | 2 x 15 | — |
| デッドバグ | 自重 | 2 x 8/側 | — |

**NG**: DB RDL で腰が丸まったら即セット中止。ヒンジは股関節から。

### Form Check（DB RDL）

- [ ] 股関節から折り畳んでいる（膝主導でない）
- [ ] 腰が丸まっていない（全ROM）
- [ ] DB が身体に近い軌道
- [ ] トップで臀筋をロックアウト
- [ ] ハムストリングのストレッチ感あり
