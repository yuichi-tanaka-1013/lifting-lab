### 採用アプローチ: トレ日/休息日で米を分ける（-500 kcal/日）

食事記録分析（2026-05-31）の結果、カロリーは既にカット水準だった。
米の量を調整するだけで目標赤字に到達する。

| パラメータ | 値 |
|-----------|-----|
| **トレ日（月火木金）** | **米 1.8合/日**（~2130 kcal） |
| **休息日（水土日）** | **米 1.2合/日**（~1800 kcal） |
| 週平均カロリー | ~2000 kcal/日 |
| カロリー赤字 | **-500 kcal/日**（TDEE ~2500 ベース） |
| タンパク質 | ~147g/日（トレ後にプロテイン1スクープ追加） |
| 脂肪減少ペース | ~0.5 kg/週 |
| 推定期間 | **約3ヶ月** |
| 目標到達 | **2026年8月末** |

### 実行すること（2つだけ）

1. **トレ日は米 1.8合、休息日は米 1.2合** — おかずは変えない
2. **トレ後にプロテイン 1スクープ追加** — タンパク質を 123g → 147g に

### 前提条件

1. トレーニングを継続する（週4回 Upper/Lower）
2. クレアチン継続（カット中の筋量維持に寄与）
3. 睡眠の確保（7時間以上が理想）
4. トレの質が落ちたら米を少し戻す（パフォーマンス優先）

### muscle memory によるリコンプ効果

出戻り組は caloric deficit の中でも筋量を回復できる可能性がある:
- 骨格筋量のギャップがわずか 1.6 kg
- [[Muscle Memory]] の筋核永続性により、再成長のシグナルが強い
- → カット中でも +1.6 kg の筋量回復が並行して起こる可能性（真のリコンプ）
