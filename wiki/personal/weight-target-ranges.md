# Weight Target Ranges — 減量末期とバルク期の体重設計

> 身長 181.6 cm。ピーク時 LBM 71.6 kg（SMM 40.9 kg）を基準に算出。
> **InBody 測定時に「今どこにいるか」「どこに向かうか」を判断するための地図。**

## 結論レンジ

| フェーズ | 体重 | 体脂肪率 | LBM 想定 | 用途 |
|---------|------|---------|---------|------|
| 減量末期（攻めの絞り） | 78–80 kg | 10–12% | 71–72 kg | 撮影・夏前など短期目的 |

## 算出根拠

### 1. LBM 天井の推定

- ピーク（2023-09）: LBM 71.6 kg, SMM 40.9 kg, BF 14.5%, BMI 25.6
- 正規化 FFMI: **約 21.6** → ナチュラル上限（~25）にはまだ余裕
- 継続的なトレで **LBM 73–75 kg（+2–3 kg）** までは現実的（muscle memory + 純粋な新規ゲイン）
- 40 代の現実として LBM 73–74 kg がプラトーラインになる可能性

### 2. 体脂肪率の意味

| BF% | 見た目 | 持続性 |
|-----|--------|--------|
| 8–9% | ステージ仕上げ、血管浮き出る | × 数週間限界 |
| 10–12% | 4–6 pack、肩の細かい筋繊維 | △ 短期間 |
| 13–15% | 腹筋うっすら、健康診断オール優 | ◎ 持続可能 |
| 16–18% | 服を着ると体つきいい、脱ぐと薄ぼんやり | ◎ バルクに有利 |
| 20%+ | お腹に乗る | × バルク限界超え |

### 3. 計算

```
LBM = 71.6 kg（ピーク）の場合:
  10% BF → 71.6 / 0.90 = 79.6 kg
  12% BF → 71.6 / 0.88 = 81.4 kg
  14% BF → 71.6 / 0.86 = 83.3 kg
  16% BF → 71.6 / 0.84 = 85.2 kg

LBM = 74 kg（+2.4 kg、現実的な伸びしろ）の場合:
  12% BF → 74 / 0.88 = 84.1 kg
  15% BF → 74 / 0.85 = 87.1 kg
  17% BF → 74 / 0.83 = 89.2 kg
```

## 出戻り組としての示唆

### 直近のカット目標（2026 年内）

- 着地点: **83–84 kg @ 14–15%**（ピーク再現ライン）
- ここから先（80 kg 切り）は明確な目的（撮影・コンテスト等）がない限り推奨しない
- 関連: [[Body Composition Timeline]] のカット計画（-500 kcal/日、2026 年 8 月末目標）

### 長期視点（2027 年以降のバルク）

- **バルク上限 88 kg を絶対超えない**
  - 88 kg × 17% BF = 体脂肪 15 kg。ピーク時 12.1 kg + 3 kg まで
  - 超えると以下のリスク：
    1. 内臓脂肪の再加速（体幹に脂肪 61% 集中する体質傾向）
    2. ウエスト 95 cm 超え
    3. 腰痛リスク再燃
- バルク → カットの幅は **±6–8 kg まで**
- ±10 kg を超えるサイクルはホルモン環境・インスリン感受性を悪化

### BMI の扱い

- BMI 25.6（ピーク時）は WHO 基準で「過体重」だが体組成的には健康
- 健康診断の「適正体重 73 kg」は無視してよい
- 代わりに以下を健康指標として使う：
  - **WHR ≤ 0.90**
  - **ウエスト ≤ 90 cm**
  - 内臓脂肪レベル ≤ 10

## 年間アンカー2本: 4月健診 + 8月上旬の海（2026-07-22 設計、同日改訂）

> 本人要望: **(1) 毎年4月の健康診断時にいい状態で臨みたい、(2) 8月上旬に海に行くため絞れた状態でいたい**。
> この2点を年間サイクルの固定アンカーとし、バルク/カットの切れ目を両方に合わせる。

### 年間サイクル（確定版・2027年から適用。本人決定 2026-07-22: 8月ピーク = 80-81kg / BF 11-12%）

```
10月 ─────── 2月中旬   バルク（83-84 → ~86 kg）★年間唯一の増量ブロック
2月中旬 ──── 3月末     ミニカット 6週（86 → 84.5-85 kg / BF ~15%）
【4月上旬】           健康診断 = カロリー赤字明けの良い状態で受診
4月 ──────── GW       メンテ（84.5 kg 前後を維持、健診明けの回復）
GW明け ───── 7月末     本カット 12-13週（84.5 → **80-81 kg / BF 11-12%**）
                      ・ペース -0.3〜-0.4 kg/週（終盤は自然に鈍る前提）
                      ・6月中旬に1週間のダイエットブレイク（維持カロリー）を挟む
【8月上旬】            海 = 年間ピーク（6パックの影〜明確、上位数%ライン）
8月 ──────── 9月中旬   リバースダイエット（2週ごと +100-150 kcal、82 kg 前後まで戻す）
9月下旬〜10月         バルク再開へ
```

- 4月時点の目標値: **体重 84-85 kg / BF 14-15% / ウエスト ≤ 90 cm / WHR ≤ 0.90 / 内臓脂肪 ≤ 7**
- **8月上旬の目標値: 80-81 kg / BF 11-12%（本人決定）**。腹筋 4パック明確〜6パックの影、腕・肩の血管が出るライン
- **トレードオフ（本人了承の上で採用）**: 旧2アンカー案にあった「4-6月のバルク後半」を廃止し、年間の増量は 10月-2月の約4.5ヶ月に集約。LBM ゲイン見込みは +2.5-3.5 → **+1.5-2.5 kg/年** に低下し、[[Strength Roadmap]] の上級ライン到達は約半年〜1年後ろ倒し（2029年想定）
- カットの数値検証: 84.5 kg/15% → 80.5 kg/11.5% は脂肪 -3.4 kg。12-13週 × -0.3 kg/週 ≒ -4 kg で**ブレイク込みでも余裕を持って到達可能**
- 深いカット終盤（12% 台）の挙上は「維持できれば成功」。減少 5% 以内は正常範囲
- リバースダイエットを省くと秋のリバウンドで脂肪から戻るため必須（8-9月に急いで食べ戻さない）

### 健診直前の実務メモ

| 項目 | 対応 |
|------|------|
| 採血前 2-3日 | 高強度トレをオフにする（筋トレ直後は AST/ALT/CK が一過性に上昇し、肝機能異常と誤読されうる） |
| クレアチン | 継続でよいが、**クレアチニン値が高めに出やすい**ことを認識しておく（腎機能異常ではなくサプリ由来の場合が多い。再検査になったら申告） |
| 高タンパク食 | BUN（尿素窒素）が高めに出やすい。同上 |
| 前日 | 飲酒なし・睡眠 7h（普段通りでOK） |

## InBody 測定時のチェックリスト

測定値が出たら以下を確認：

1. **体重の位置**：上記レンジのどこにいるか
2. **LBM**：前回からの増減（カット中なら維持できているか、バルク中なら増えているか）
3. **体脂肪率**：レンジ表の現実的なゾーンに収まっているか
4. **ウエスト・WHR**：BF% より体感に近い指標。≤ 90 cm / ≤ 0.90 を死守
5. **内臓脂肪レベル**：体幹脂肪集中タイプなので最重要監視項目
6. **次フェーズの判断**：
   - メンテ目標到達 → バルク開始 OR 維持継続
   - バルク上限到達 → カット開始（先延ばしにしない）
   - カット末期で停滞 → 体組成より体感（睡眠・気分・トレパフォーマンス）を優先
